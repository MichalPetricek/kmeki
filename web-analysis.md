## 6. Technicka analyza

### 6.1 CMS a technologie
- **CMS:** Redakcni system IPO od ANTEE s.r.o.
- **Framework:** Nette (cesky PHP framework) - pro formulare a validaci
- **Analytika:** Matomo (byvaly Piwik) na matomo.antee.cz, site ID 1292
- **Anti-spam:** Google reCAPTCHA
- **JS knihovny:** jQuery, Select2 (filtrovani tagu v galeriich)
- **Cookies:** Matomo ma cookies zakazane, vlastni cookie consent banner

### 6.2 Obrazky
Vsechny obrazky jsou servrovany pres dynamicky endpoint `/image.php` s parametry:
- `nid` - node ID (13887 pro cely web)
- `oid` - object ID (unikatni pro kazdy obrazek)
- `width` / `height` - volitelne rozmery

### 6.3 Sledovani
- Sledovani kliknuti na mailto: odkazy (Goal ID 4 v Matomo)
- Sledovani kliknuti na tel: odkazy
- Custom variables pro typ kliknuti

### 6.4 SEO problemy
- Chybi meta description na hlavni strance
- Mnoho galeriových obrazku nema alt texty
- Duplikovany obsah mezi CZ a EN verzemi na nekterych strankach
- Preklep v URL anglicke verze referenci (/references/architecture-and-onstruction)
- Preklep v textu ceske verze ("archtektonicke" misto "architektonicke")
